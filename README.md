# AI Assistance Chat

Простой веб-чат с AI-ассистентом для общения и хранения диалогов.

## Что умеет

- создавать и продолжать чаты;
- хранить историю сообщений в базе данных;
- показывать список чатов пользователя;
- работать через удобный интерфейс с отдельным окном диалога.

## Архитектура

- server/ — backend (FastAPI + MongoDB + интеграция с LLM);
- client/ — frontend (React + TypeScript).
