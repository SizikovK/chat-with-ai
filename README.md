# Simple AI-Agent


Что реализовать:
- откправка сообщения с клиента
- получение сообщения на сервере
- обработка
- отправка сообщения с сервера
- получение сообщения на клиенте

- получение json всей переписки с сервера и скачивание файла на клиенте
- загрузка json всей переписки на клиенте

- история чатов по chat id

- отправка формы при регистрации
- валидация
- запись в бд

- отправка формы при входе
- валидация
- вход




```bash
uvicorn app.main:app --reload
```

Создание пользователя:

```bash
curl -X POST http://127.0.0.1:8000/auth/users \
  -H "Content-Type: application/json" \
  -d '{
    "nickname": "ivan",
    "email": "ivan@example.com",
    "password": "secret123"
  }'
```
