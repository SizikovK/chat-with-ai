# Neuro Chat (Frontend)

Фронтенд чата на React + TypeScript.

## Что реализовано

- регистрация и вход пользователя;
- защищенные маршруты (`/chats`, `/chats/:chatId`, `/profile`);
- список чатов, окно сообщений, отправка сообщений, удаление чатов;
- глобальное состояние через Redux Toolkit (`auth`, `chats`, `messages`).

## Технологии

- React
- TypeScript
- React Router
- Redux Toolkit
- Vite
